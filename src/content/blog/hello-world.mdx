## A quick taste

Here's the kind of thing I find worth writing down — a tiny helper I reach for
constantly:

```ts
export function groupBy<T, K extends PropertyKey>(
  items: readonly T[],
  key: (item: T) => K,
): Record<K, T[]> {
  return items.reduce((acc, item) => {
    const k = key(item);
    (acc[k] ??= []).push(item);
    return acc;
  }, {} as Record<K, T[]>);
}
```

Nothing fancy — but the `??=` there saves an `if` check every single time.
